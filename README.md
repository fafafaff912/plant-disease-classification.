#  Классификация состояния растений по снимкам

## Описание задачи

Многоклассовая классификация заболеваний растений по изображениям листьев.
Модель определяет — здоровое растение или поражено конкретным заболеванием
(бактериальным, грибковым, вирусным).

## Цель и мотивация

- Раннее обнаружение болезней растений для снижения потерь урожая
- Автоматизация мониторинга посевов (применимо к данным с дронов)
- Замена ручного осмотра на AI-решение

## Данные

**PlantVillage Dataset** — 54 303 изображения листьев, 38 классов.

- Источник: [Kaggle](https://www.kaggle.com/datasets/emmarex/plantdisease)
- 15 видов растений × здоровые + различные болезни
- Разделение: 80% train / 10% val / 10% test (стратифицированное)

## Архитектура модели

**EfficientNet-B0** с transfer learning (ImageNet).

Обоснование выбора:

| Модель | Параметры | ImageNet Top-1 | FLOPs |
|--------|-----------|----------------|-------|
| ResNet-50 | 25.6M | 76.1% | 4.1G |
| **EfficientNet-B0** | **5.3M** | **77.1%** | **0.39G** |
| ViT-Base | 86M | 77.9% | 17.6G |

EfficientNet-B0 — лучший баланс точности и скорости, что важно
для развёртывания на edge-устройствах (дроны).
