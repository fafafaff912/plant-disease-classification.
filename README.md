# Классификация состояния растений по мультиспектральным снимкам с дронов

## Краткое описание
Проект решает задачу бинарной классификации состояния растений (здоровое / больное) на основе мультиспектральных изображений, полученных с беспилотных летательных аппаратов. В качестве демонстрационных данных используется публичный датасет PlantVillage (RGB-изображения), однако архитектура легко адаптируется для многоканальных снимков.

## Цель и мотивация
Автоматизация мониторинга посевов позволяет своевременно выявлять заболевания и стресс растений, минимизировать потери урожая и оптимизировать применение удобрений. Разработанная модель может быть интегрирована в систему агромониторинга на базе дронов.

## Использованные данные
- **PlantVillage Dataset** (https://www.kaggle.com/datasets/emmarex/plantdisease) — содержит около 50 000 размеченных изображений листьев здоровых и больных растений (RGB, 224x224).
- *Примечание:* для работы с реальными мультиспектральными данными потребуется модификация первого слоя сети.

## Архитектура модели и обоснование
- Базовая модель: **ResNet18** (предобученная на ImageNet).
- Последний полносвязный слой заменён на новый с количеством выходов, равным числу классов.
- **Обоснование:** ResNet обеспечивает хороший баланс между точностью и вычислительной сложностью, предобучение ускоряет сходимость на ограниченном объёме данных. При необходимости модель может быть заменена на EfficientNet или Vision Transformer.

## Метрики качества
- Accuracy (точность)
- F1-score (взвешенный)
- Confusion matrix (матрица ошибок)

## Результаты
После обучения на 10 эпохах с валидационной долей 20% были получены следующие показатели (пример):
- Accuracy: 0.9823
- F1-score (weighted): 0.9824
- Матрица ошибок сохраняется в файл `reports/confusion_matrix.png`.
